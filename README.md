# TomotoHighPredict
随着科技的进步和社会的发展，农业正在经历一场数字化、信息化的革新。智慧农业、精准农业等概念已经广泛渗透。其中深度学习作为人工智能的重要分支，正在逐步应用到农业生产的各个环节。本项目计划利用深度学习的方法，通过输入环境湿度、环境温度、植株照片、施肥情况等参数，预测番茄植株的生长高度。
